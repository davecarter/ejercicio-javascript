## Calculadora JavaScript

En este ejercicio pedimos 2 valores numéricos al usuario mediante la función [prompt()](https://developer.mozilla.org/en-US/docs/Web/API/Window/prompt) realizaremos una suma y devolveremos el resultado por pantalla y en la consola de JavaScript.

### Ejercicio:
- Cambia la operación aritmética de SUMA a MULTIPLICACIÓN.
- Añade todos los cambios necesarios y haz un `commit` con el mensaje: 'refactor de función SUMA a MULTIPLICACION'
- La calculadora ahora ha de Multiplicar los valores introducidos por el usuario y mostrar el resultado tanto en la consola de JavaScript como en Pantalla.
